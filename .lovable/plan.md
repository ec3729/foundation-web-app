

## Simple Landing Page

A clean, minimal single-page landing page to serve as a foundation for future features:

- **Hero section** with a centered heading, short subtitle, and a call-to-action button
- **Clean, modern design** using the existing design system (neutral tones, clean typography)
- **Responsive layout** that looks good on mobile and desktop
- **Footer** with a simple copyright line

This gives you a solid starting point to layer on more sections, navigation, and features over time.
